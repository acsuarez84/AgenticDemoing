How the two systems work
LEFT — Text Position: Each word carries a number that is its actual byte location in the original 19th-century source text. The word "silk" first appears at byte 46 in Gilroy's 1845 manuscript. "Arachne" sits at byte 16,383. Words never written in the source texts — like "erased" or "silenced" — carry the value 0. These positions feed the algorithm that shapes each textile pattern: the number determines ring spacing, spiral tightness, wave frequency, or ray count.
RIGHT — ASCII Code: ASCII (American Standard Code for Information Interchange) assigns every letter a fixed number: A=65, B=66, a=97, b=98, and so on. The algorithm sums all character codes in a word — "silk" becomes s(115)+i(105)+l(108)+k(107)=435. Unlike text positions, these values are universal: "silk" always equals 435 regardless of which book it appears in. The same word produces the same number everywhere.
The difference matters: Text positions are contextual — the same word in a different book produces a different pattern. ASCII sums are abstract — the word's internal spelling alone determines the shape. When both systems generate simultaneously from the same rhetoric, you see how context versus abstraction weaves different textiles from identical language.

How points connect: Each letter in the 3D space occupies a position determined by its word's seed value (text position or ASCII sum). When two letters from different rhetorical categories — or from different generation layers — fall within 0.7 world units of each other in 3D space, they form a connection. The white lines at these intersections grow thicker with each generation as more threads accumulate. The density of connections reveals where different centuries of rhetoric about women physically converge — where "silk" from 1845 meets "spinning" from 1914, where divine origin stories collide with anonymous labor. Touch any point and drag it — the poem below will rewrite itself from the words you disturb.
Why 5 connections per generation: Each new generation of the weaving pattern is limited to exactly 5 cross-connections — five threads that bridge across the shape's rows, evenly spaced through the textile. This constraint mirrors how actual weavers work: a loom passes the shuttle across a finite number of warp threads per row, not through every possible intersection at once. Each generation adds its own 5 connections on top of the previous layers, so by generation 4 there are 20 cross-threads accumulating, by generation 10 there are 50. The visual slowly thickens from sparse to dense — from recognizable shape to unrecognizable cloth — the same way language about women in these texts shifted from named goddesses to anonymous labor over centuries. Only the most connected points — the intersections with the highest number of overlapping threads — trigger new poems and new pattern generations. A single point where "silk" and "shame" and "goddess" all converge carries more generative weight than a hundred isolated letters.
